|-------|----------|

**גודל מומלץ:** התמונות שלך ושל צחי 400×400 פיקסלים או יותר (ייחתכו לעיגול אוטומטית). הלוגו רצוי PNG עם רקע שקוף.

### רקע וגאס

הדף משתמש בתמונת וגאס מ-Unsplash שעובדת מצוין כברירת מחדל. אם תרצה תמונה ספציפית משלך:

```css

```


ושמור את התמונה בשם `vegas-bg.jpg` באותה תיקייה. רוחב מינימלי 1920px לאיכות טובה.

**טיפ:** Unsplash, Pexels, או Pixabay מציעים תמונות וגאס איכותיות בחינם וללא הגבלה מסחרית.

## שלב 3: חבר את הטופס לקבל את הנתונים

כרגע הטופס מציג הודעת הצלחה אבל לא שולח את הנתונים לשום מקום. יש 3 אפשרויות, מהקלה לחזקה:

### אפשרות A: Google Forms (הכי מומלץ למתחילים — חינם, מנהל הכל)

1. לך ל-`forms.google.com` ויצור טופס חדש עם השדות הבאים (סדר חשוב):
   - שם פרטי (טקסט)
   - שם משפחה (טקסט)
   - אימייל (טקסט)
   - טלפון (טקסט)
   - אינסטגרם (טקסט)
   - דיסקורד (טקסט)
   - גיל (טקסט)
   - מדינה (טקסט)
   - אישור כניסה לארה"ב (בחירה: ESTA / ויזה / דרכון אמריקאי / אין)
   - אישור 8 שלבים (תיבת סימון: כן)
   - הסכמה לתקנון (תיבת סימון: כן)
   - הסכמה לשיווק (תיבת סימון: כן)

2. לחץ "שלח" → אייקון `<>` (הטמעה) → תקבל לינק.

3. כדי להגדיר את הטופס שלנו לשלוח אליו, יש שיטה פשוטה: כל שדה גוגל יש לו `entry.XXXXX`. תצטרך לפתוח את הטופס בתצוגה מקדימה, ללחוץ Inspect Element, ולמצוא את ה-IDs.

**אבל יש דרך הרבה יותר פשוטה:**

### אפשרות B: Formspree (הכי מהיר — חינם עד 50 הרשמות בחודש, $10 לחודש לאחר מכן)

1. הירשם ב-`formspree.io` (חינם).
2. צור Form חדש → תקבל URL בצורה: `https://formspree.io/f/XXXXXXXX`
3. בקובץ `index.html` חפש את השורה:
   ```html
   <form id="raffle-form" novalidate>
   ```
   החלף ל:
   ```html
   <form id="raffle-form" novalidate action="https://formspree.io/f/XXXXXXXX" method="POST">
   ```
4. בקובץ `index.html` בקטע ה-JavaScript, חפש את ה-`form.addEventListener('submit', ...)` ופשוט מחק את כל ה-block שלו, או החלף ב:
   ```js
   form.addEventListener('submit', function (e) {
     // נותנים לטופס לשלוח כרגיל ל-Formspree
   });
   ```
5. הנתונים יגיעו ישירות למייל שלך.

### אפשרות C: Tally / Typeform (חוויית הרשמה מובנית)

חלופה: השתמש ב-`tally.so` (חינם) או `typeform.com` (יוקרתי) ושים את הקישור ככפתור CTA במקום הטופס בדף. מתאים אם אתה רוצה חוויית הרשמה אינטראקטיבית יותר.

**ההמלצה שלי: Formspree.** דקה להגדיר, הנתונים מגיעים למייל, אפשר לייצא ל-CSV.

## שלב 4: העלה לאוויר (3 דקות)

הדף הוא קובץ HTML בודד אז כל שירות אחסון סטטי יעבוד. הכי מהיר:

### Netlify Drop (חינם, ללא הרשמה צריך, מקבלים URL מיידי)

1. לך ל-`app.netlify.com/drop`
2. גרור את התיקייה שמכילה את `index.html` (ואת התמונות אם החלפת)
3. תקבל URL כזה: `https://random-name.netlify.app`
4. אם תרצה — אפשר לשנות לשם נחמד יותר ב-Site Settings → Domain → Change site name.

### דומיין משלך

אם יש לך דומיין (למשל `realvardi.com/vegas`), Netlify מאפשר לחבר אותו בחינם תחת Site Settings → Domain.

**טיפ עבור הקמפיין הממומן:**
- צור URL קצר וקליט: `realvardi.com/vegas` או `realvardi.com/win`.
- הוסף UTM פרמטרים בלינק שאתה שם באינסטגרם (לדוגמה `?utm_source=instagram&utm_medium=paid&utm_campaign=vegas2026`) כדי לעקוב מאיפה הגיעו הלידים.

## שלב 5: פיקסל פייסבוק / מטא (לקמפיין הממומן)

חיוני אם אתה מריץ קמפיין ממומן באינסטגרם — תוכל לעקוב אחר conversions ולעשות retargeting.

1. לך ל-Meta Business Manager → Events Manager → קבל את קוד הפיקסל.
2. בקובץ `index.html` הוסף לפני `</head>` את הקוד שמטא נתנו לך (זה נראה כמו `fbq('init', 'XXXXX')`).
3. הקוד שלי כבר כולל את השורה `fbq('track', 'Lead')` שתופעל אוטומטית כשמישהו מסיים הרשמה. אז ברגע שתדביק את הפיקסל, האירוע יירשם.
